## EVIDENCE CLASSIFICATION SYSTEM

All findings are classified using a 4-tier system:

### CLASS A (SUBPOENA-READY)
- **Confidence:** 90-100%
- **Verifiable:** Blockchain transactions with hashes
- **Court-ready:** Yes, irrefutable evidence
- **Examples:** Large ETH transfers, Stake.com deposits

### CLASS B (CORROBORATED)
- **Confidence:** 75-89%
- **Verifiable:** Multiple sources confirm pattern
- **Court-ready:** Strong circumstantial evidence
- **Examples:** Multi-chain patterns, wallet clustering

### CLASS C (HYPOTHESIS-STRONG)
- **Confidence:** 60-74%
- **Verifiable:** Pattern suggests conclusion
- **Court-ready:** Requires additional analysis
- **Examples:** Round-tripping, wash trading

### CLASS D (EXPLORATORY)
- **Confidence:** 40-59%
- **Verifiable:** Requires off-chain data (KYC, IP logs)
- **Court-ready:** Not without corroboration
- **Examples:** Wallet attribution, identity linking
